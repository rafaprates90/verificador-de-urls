# 🔗 Verificador de URLs PK CÍVEL

API para verificação pública de URLs no Yahoo ou Bing, com geração de screenshots da busca como evidência.

## 🚀 Funcionalidades
- Verifica se a URL está localizada no buscador.
- Gera screenshot da página da busca.
- Retorna status, buscador, link do screenshot e data/hora.

## 🛠️ Como Rodar na Render
1. Crie conta em https://render.com
2. Crie um novo Web Service.
3. Escolha **“Use Docker”**.
4. Faça upload dos arquivos (`main.py`, `Dockerfile`, etc.).
5. Configure a porta como **10000**.
6. Gere sua URL pública.
